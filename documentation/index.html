CSS File Structure

We are using 9 css files which are seperated to extend usability and customization. If you are going to use this theme, we recommend to merge all the css into one file to reduce http queries.

<!-- Bootstrap CSS -->
<link rel="stylesheet" href="css/assets/bootstrap.min.css">
<!-- Font Awesome -->
<link rel="stylesheet" href="css/assets/font-awesome.min.css">
<!-- Owl Carousel -->
<link rel="stylesheet" href="css/assets/owl.carousel.css">
<link rel="stylesheet" href="css/assets/owl.transitions.css">
<!-- Normalize CSS -->
<link rel="stylesheet" href="css/normalize.css">
<!-- Magnific Popup -->
<link rel="stylesheet" href="css/assets/magnific-popup.css">
<!-- Custom CSS -->
<link rel="stylesheet" href="css/style.css">
<link rel="stylesheet" href="css/assets/responsive.css">

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

Font Used

We have used three types of font throughout the template.

These are:

1. Roboto - Heading font

2. Dosis - Body Font

3. Sacramento - Signature Font

<!-- Google Fonts -->
<link href="https://fonts.googleapis.com/css?family=Roboto:300,400,500,700" rel="stylesheet">
<link href="https://fonts.googleapis.com/css?family=Dosis:300,400,500" rel="stylesheet">
<link href="https://fonts.googleapis.com/css?family=Sacramento" rel="stylesheet">

All of the fonts are available on google fonts. You can choose other fonts from google directory. And link them at head

Logo Change

It's very easy to change your logo. Just take a look at the code below. Change the logo link to your own image or text logo.

<section class="intro" id="intro">
    <div class="container">
        <div class="row">
            <div class="col-sm-12">
                <div class="logo">
                    <a href=""><img src="images/logo.png" alt=""></a>
                </div><!-- Ends: .logo -->
            </div>
